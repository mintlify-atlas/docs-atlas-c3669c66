Social commands let you interact with other users through fun roleplay actions accompanied by anime-style GIF images. These commands fetch images from popular anime GIF APIs to make your interactions more expressive.

<Note>
  All social commands display anime-style GIF images from waifu.pics and nekos.life APIs.
</Note>

## Bite

Bite someone playfully.

### Syntax

```bash
/bite user:<user> [reason:"optional text"]
```

### Parameters

<ParamField path="user" type="User">
  The user you want to bite
</ParamField>

<ParamField path="reason" type="String">
  Optional roleplayable reason for the action
</ParamField>

### Example

```bash
/bite user:@friend reason:"for stealing my snacks"
```

## Usage tips

<AccordionGroup>
  <Accordion title="Image sources">
    Images are randomly fetched from:
    - [waifu.pics](https://waifu.pics) - High-quality anime GIFs
    - [nekos.life](https://nekos.life) - Additional anime reaction images
    
    Some commands (hug, kiss, pat, slap, smug) use both APIs for variety.
  </Accordion>

  <Accordion title="Error handling">
    If the image API is temporarily unavailable, you'll see an error message suggesting you:
    - Try again later
    - Join the [Support Server](https://discord.gg/mm7Ke8T) if the issue persists
  </Accordion>

  <Accordion title="Interactive roleplay">
    These commands are perfect for:
    - Creating a fun, engaging server atmosphere
    - Expressing emotions and reactions
    - Building community through playful interactions
    - Adding personality to conversations
  </Accordion>
</AccordionGroup>

## Troubleshooting

<Warning>
  If images fail to load:
  - The external API might be temporarily down
  - Check your internet connection
  - Try the command again after a few moments
  - Report persistent issues in the [Support Server](https://discord.gg/mm7Ke8T)
</Warning>
